# Astro Vision Analyzer  (Mistral Hackathon 2026)

Project Overview:
I built a Python tool that identifies planets from images and calculates personalized space statistics. It combines AI-powered vision with custom physics logic.

What I implemented:

- AI Vision: Connected to Mistral's Pixtral-12B model via Hugging Face API to analyze planetary features.

- Space Physics Logic: Programmed Python functions to calculate weight and age on other planets based on real astronomical data.

- Survival Analysis: Prompted the AI to simulate human survival scenarios based on atmospheric pressure and temperature.

- Security Best Practices: Managed sensitive API tokens using Environment Variables (.env) to keep the code secure.

Key Learnings:

- Integrating Multimodal AI (Images + Text).

- Handling JSON responses and API errors.

- Structuring Python code with functions for better readability.
